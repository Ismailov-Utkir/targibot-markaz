# Targ'ibot Markazi — Hududiy Monitoring

Raqamli targ'ibotni muvofiqlashtirish bo'limi uchun boshqaruv tizimi.

## Foydalanuvchilar

- **Administrator** — parol bilan kiradi, postlarni ko'radi, statistika kiritadi
- **Vakil** (14 hudud) — username + PIN bilan kiradi, postlarini yuboradi

## Funksiyalar

- Vakillar Telegram / Instagram / Facebook postlarini havola bilan yuboradi
- Admin har platformadan ko'rishlar, layk, kommentni qo'lda kiritadi
- Dashboard: platforma bo'yicha umumiy statistika va o'sish grafigi
- Statistika, filtrlar, CSV eksport, JSON zaxira

## Texnologiyalar

- HTML / CSS / JavaScript (statik)
- Chart.js — o'sish grafigi
- localStorage — ma'lumotlar brauzerda saqlanadi

> Eslatma: ma'lumotlar har bir brauzerda alohida saqlanadi. Umumiy baza uchun Supabase qo'shilishi mumkin.
